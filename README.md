# 🐍 Exercícios de Python - CodeMayPy

Este repositório contém uma coleção de mais de 80 exercícios práticos resolvidos por mim durante meus estudos de Python no Curso em Vídeo. O objetivo é consolidar os fundamentos da linguagem, desde lógica básica até estruturas de dados mais complexas.

## 🧠 Meu Caderno de Revisão Dinâmico
Mais do que apenas um repositório de códigos, este projeto funciona como o meu **caderno digital de estudos**. 
* **Evolução Contínua:** Estou sempre revisitando códigos antigos para refatorar, aplicar novas soluções mais eficientes e corrigir pequenos detalhes.
* **Resumos Pessoais:** Além dos desafios resolvidos, você encontrará guias visuais e explicações didáticas feitas por mim nas pastas de tópicos importantes (como `for`, `while`, `Tuplas` e `Listas`), servindo como minha principal fonte de consulta técnica.

## 🚀 Tecnologias Utilizadas
* **Python**
* Python 3.
* Git & GitHub (Boas práticas com Conventional Commits).
* Markdown (para documentação e resumos).
* Lógica de Programação.

## 📂 Estrutura do Repositório
O projeto está organizado por módulos de aprendizado (Mundos), espelhando a cronologia do curso. Dentro de cada pasta, os exercícios seguem o padrão sequencial (ex001.py, ex002.py, etc.) acompanhados de arquivos de revisão.

- **Mundo_Um**: Fundamentos da linguagem, variáveis, tipos primitivos, operadores aritméticos, utilização de módulos e estruturas condicionais simples/compostas.

- **Mundo_dois**: Estruturas de controle, condições aninhadas em cadeia e estruturas de repetição (for e while com break/continue).

- **Mundo_três**: Estruturas Compostas, pastas dedicadas ao tratamento avançado de dados com Listas, Tuplas e Dicionários.
